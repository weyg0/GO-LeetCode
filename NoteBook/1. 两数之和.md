## 1. 两数之和

时间复杂度`O(n)`，空间复杂度`O(n)`。

顺序扫描数组，用`map`的`key`保存数组的值、`value`保存数组的下标，遍历数组的过程中发现`map`存有相加等于目标值的`key`就返回两个数字的下标，不存在则将该数的值-下标对存入`map`。

```go
func twoSum(nums []int, target int) []int {
	m := make(map[int]int)
	for i := 0; i < len(nums); i++ {
		if j, ok := m[target-nums[i]]; ok {
			return []int{j, i}
		}
		m[nums[i]] = i
	}
	return nil
}
```
